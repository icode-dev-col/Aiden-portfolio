# Aiden's Portfolio

A creative, responsive portfolio website built to showcase my web design and development projects. I am a student of the Orange Belt program at iCode Columbia!

## Tools & Features Explained

- **HTML5 Canvas & JavaScript (Interactive Background)**
  HTML Canvas is like a digital drawing board. JavaScript acts like an invisible artist that constantly draws and moves our gaming shapes around! This makes the background look alive when you scroll down.
  
- **CSS Animations (RGB Gaming Text)**
  CSS animations are instructions that tell colors to move over time. We used this to paint a rainbow that slides across the words "Hi I'm Aiden," making it look like a glowing light-up gaming keyboard!

- **CSS Custom Variables (Theme Toggle)**
  Variables are like magic buckets where we store our favorite colors. When you click the Sun or Moon button, JavaScript flips the buckets to change the whole website from light mode to dark mode instantly.

- **CSS Flexbox & Hover Effects (Fancy Project Cards)**
  Flexbox is a tool that perfectly lines up boxes on a page, so the project tiles look neat even on cell phones. The hover effects make the tiles pop up and blur the background when your mouse touches them!

- **JavaScript Intersection Observer (Scroll Reveal Animation)**
  This JavaScript tool acts like a lookout tower perfectly watching your screen. When you scroll down to a new section, it spots the elements and instantly tells CSS to glide them upwards into view smoothly!

- **CSS (Live Dragon Avatar)**
  We wrapped the dragon in complex CSS keyframes to simulate continuous floating and breathing. Finally, we engineered a custom Javascript particle system that continuously breathes out an animated stream of cute purple fire particles that move in sync with the dragon's body!

## Setup and Development

Since this project avoids compiling stages and runs on pure frontend technologies, it is incredibly simple to set up and run. 

1. **Clone the repository:**
   ```bash
   git clone https://github.com/VNintendo/Aiden-portfolio.git
   cd Aiden-portfolio
   ```

2. **Launch locally:**
   Open the `index.html` file directly in your web browser of choice.
